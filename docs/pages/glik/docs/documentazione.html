Introduzione

Glik è un visualizzatore desktop per il monitoraggio della glicemia tramite Nightscout, che permette di tenere sotto controllo i valori glicemici direttamente dal computer.

Caratteristiche Principali

  • Monitoraggio in tempo reale dei valori glicemici
  • Notifiche desktop personalizzabili
  • Visualizzazione grafica dei trend
  • Integrazione con Nightscout
  • Avvio automatico con Windows
  • Interfaccia minimale nella system tray

Installazione

Requisiti di Sistema

  • Windows 10/11 (64-bit)
  • 100MB spazio su disco
  • 2GB RAM
  • Connessione Internet
  • Sito Nightscout configurato

Supporto

Come Ottenere Supporto

Ci sono diversi modi per ottenere supporto per le applicazioni 3.6 Gen:

  • GitHub Issues: Segnala problemi o richiedi nuove funzionalità
  • Email: Contattaci direttamente per supporto personalizzato
  • Documentazione: Consulta le guide dettagliate per ogni app

Se desideri supportare lo sviluppo delle app, puoi fare una donazione tramite Ko-fi. La donazione è completamente opzionale e non influisce sull'uso gratuito delle app.

Supportami su Ko-fi

Pagnucom è un sistema per la creazione e gestione di chiavi di prodotto per software. In particolare, viene utilizzato per la gestione delle app sviluppate da Emmanuele Pani.

Inizialmente, le app sono state sviluppate con licenza MIT open source. Tuttavia, considerando il tempo e l'impegno dedicato al loro sviluppo, è stato implementato un sistema di licenze controllato. Le app rimangono open source, nel senso che è possibile contribuire al loro sviluppo, ma è necessaria una chiave di prodotto per utilizzarle.

For developers:

If you are a developer and want to contribute to the development of the apps, you can do so by forking a specific GitHub repository and sending pull requests. In this case, you do not need to request a product key through Pagnucom.

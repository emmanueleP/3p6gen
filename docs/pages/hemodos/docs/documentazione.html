Introduzione

Hemodos è un software completo per la gestione delle donazioni di sangue, progettato per semplificare il processo di prenotazione e monitoraggio delle donazioni.

Caratteristiche Principali

  • Sistema di gestione delle prime donazioni
  • Tracciamento dello stato delle donazioni
  • Statistiche dettagliate
  • Esportazione dati in formato .docx o PDF
  • Sincronizzazione cloud (OneDrive/Google Drive)
  • Backup automatico dei dati
  • Gestione multi-anno con archivio storico

Installazione

Per installare Hemodos, seguire questi passaggi:

  1. Scarica il file di installazione dal sito ufficiale
  2. Esegui il file di installazione
  3. Segui le istruzioni sullo schermo per completare l'installazione

Primi Passi

Dopo aver installato Hemodos, ecco alcuni passaggi per iniziare a utilizzarlo:

  1. Accedi al software
  2. Aggiungi un nuovo donatore
  3. Pianifica una donazione
  4. Monitora lo stato delle donazioni

Gestione Donatori

Tabella Donatori

La tabella mostra le seguenti informazioni per ogni donatore:

  • Nome: Nome del donatore
  • Cognome: Cognome del donatore
  • Prima Donazione: Indica se è la prima donazione del donatore (disabilitato di default dopo le 10:00)
  • Stato: Stato della donazione, che permette di avere un feedback all'interno delle statistiche (Non effettuata/Sì/Annullata/Rimandata)

Calendario Donazioni

Visualizzazione

  • Le date con donazioni programmate sono evidenziate in verde lime
  • Cliccando su una data si possono visualizzare/modificare le prenotazioni
  • Menu contestuale disponibile con clic destro sulla data (in sviluppo)

Funzionalità del menu:

  • Aggiungi Orario: Inserisce un nuovo slot orario per le donazioni
  • Salva: Salva le modifiche correnti
  • Elimina: Elimina l'orario selezionato
  • Esporta in DOCX: Esporta le prenotazioni del giorno in formato Word
  • Stampa: Stampa le prenotazioni del giorno selezionato

Sincronizzazione Cloud

Backup e Ripristino

  • Permette di scegliere la directory di salvataggio dei file .db principali (Locale/OneDrive/Google Drive)
  • Permette di scegliere il salvataggio automatico dell'app e la frequenza

Configurazione

Generali

  • Gestione date di donazione per anno
  • Configurazione orari disponibili
  • Personalizzazione intervalli (in sviluppo)

Gestione Anni

  • Permette di gestire gli anni e creare l'archivio (struttura) per ogni anno fino al 2099
  • Permette di eliminare l'archivio di un anno selezionato

Aspetto

  • Permette di scegliere il tema (Chiaro/Scuro)

Logo

  • Permette di scegliere l'intestazione che si visualizza nel .docx o pdf esportato.

Backup e Ripristino

Backup

Ripristino

  • Permette di ripristinare i dati da un backup
